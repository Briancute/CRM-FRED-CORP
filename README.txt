Advanced AI Integration for WordPress: Chat widget, email automation, and customer support powered by GPT.

== Description ==

WP AI Assistant brings powerful AI capabilities to your WordPress site:

* AI Chat Widget - Engage visitors with real-time AI chat
* Email Automation - Enhance your emails automatically with AI
* Customer Support - AI-powered ticket system with automatic responses

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-ai-assistant` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > AI Assistant to configure the plugin
4. Enter your OpenAI API key (required)
5. Configure which features you want to enable

== Configuration ==

1. OpenAI API Key (required)
- Get your API key from OpenAI: https://platform.openai.com/account/api-keys
- Enter it in Settings > AI Assistant

2. Feature Settings
- Enable/disable chat widget
- Enable/disable email automation
- Enable/disable customer support
- Customize initial chat message
- Choose GPT model (3.5-turbo or 4)

== Frequently Asked Questions ==

= Do I need an OpenAI API key? =

Yes, an OpenAI API key is required for all features to work. You can get one from the OpenAI website.

= Is it secure? =

Yes, the plugin follows WordPress security best practices and never stores sensitive information in plain text.
